## REMOVED Requirements

### Requirement: Ringi Objectively Certifies The Goal Through A Verification Seam
**Reason**: The MVP does not execute or certify a code change.
**Migration**: Arbitration judges dossier readiness and isolated evaluators judge fixed human conditions; neither grants approval.

### Requirement: Verification Runs Config-Supplied Commands Objectively
**Reason**: Workspace commands and code-result verification are outside the deliberation-only boundary.
**Migration**: Future executors may consume archives independently; ringi itself runs no workspace verification commands.

### Requirement: Tool Verification Outranks Model Opinion
**Reason**: The old requirement compares code tools with model completion claims.
**Migration**: Retain the authority principle structurally: process facts and archive integrity outrank NLP claims, while only humans decide the proposal.
